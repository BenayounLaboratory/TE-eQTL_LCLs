## README ##

Code for the manuscript:

Juan I. Bravo, Chanelle R. Mizrahi, Seungsoo Kim, Lucia Zhang, Yousin Suh, Bérénice A. Benayoun
"A Computational Approach for the Identification of Novel L1 Transcriptional Regulators"
(biorXiv preprint at: https://www.biorxiv.org/content/10.1101/2023.08.15.553416v2)

The code is arranged by major analysis carried out:

	# 1b_eQTL_Analysis_TElocal               	 : see README_eQTL_TElocal.txt
	# 2_RNASeq_Analysis_Top_eQTLs            	 : see README_Transcriptomics_vs_SNVs.txt
	# 2b_RNASeq_Analysis_Top_eQTLs_TElocal   	 : see README_Transcriptomics_vs_SNVs_TElocal.txt
	# 3_RNASeq_Analysis_Candidate_Gene_OE    	 : see README_Transcriptomics_Gene_OE.txt
	# 3b_RNASeq_Analysis_Candidate_Gene_OE_TElocal   : see README_Transcriptomics_Gene_OE_TElocal.txt
	# 4_RNASeq_Analysis_Peptide_Exposure     	 : see README_Transcriptomics_Peptide_Treatment.txt
	# 4b_RNASeq_Analysis_Peptide_Exposure_TElocal    : see README_Transcriptomics_Peptide_Treatment_TElocal.txt
	# 5_GWAS_Analyses                        	 : see README_GWAS.txt
